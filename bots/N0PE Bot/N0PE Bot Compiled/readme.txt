N0PE Bot Install Tutorial

- Config Datei überarbeiten(Webpanel/system/config.php)
- WebPanel Ordner auf dein Webspace / vServer / Root übertragen
- Chmod 777 auf den Webpanel Ordner & Unterordnern
- Neue DB in Mysql erstellen
- SQL Datei "db.sql" aus dem Ordner Database einspielen
- Server.exe via Builder erstellen

Config:
|| ADMIN_PASSWORD = Webpanel Login Password
|| AUTH_CODE = Authentication Code (Muss genau derselbe wie im Builder sein !)

Bot braucht bis zu 1 Minute zum connecten !
